Softwares de Versionamento
Sistema que tem por objetivo realizarm o controle de versões de um código ou documento. É possível gerenciar as versões de um código ou documento realizadas por um desenvolvedor ou equipe de desenvolvedores.

Tipos de Software de Controle de versão

 Centralizado
Que possui um repósitorio central em um servidor com controle de versões
Software Subversion


Distribuído
Possui um repositório local com controle de versões.

GIT
Git é um software livre de versionamento que surgiu para o desenvolvimento do Kernel do Linux, foi projetado e desenvolvido por Linus Torvalds em 2005. Cada diretório de trabalho do Git é um repositório com um histórico completo e habilidade total de acompanhamento das revisões, não dependente de acesso a uma rede ou a um servidor central.
